# ЕmployeeManagerMVC

The EmployeeManagerMVC is a user-friendly web application designed to streamline the management of personnel records. Built using the Model-View-Controller architecture, the application provides a structured and scalable approach to handling employee data while maintaining a clear separation between business logic and user interface.


- Features
* Add new employees to the database.
* View a list of all current employees.
* Edit employee details.
* Remove employees from the system.

# Tech Stack & Versions
* Framework: ASP.NET Core 8.0 (MVC)
* Language: C# 8
* Database: SQL Server-SQLClient
* ORM: Entity Framework Core 8.0
* Frontend: Bootstrap 5.0;

# How to run it
1. Clone the repository: `git clone https://github.com/GeorgiPasev347/EmployeeManagerMVC.git`
2. Update the connection string in `appsettings.json` to match your local SQL Server.
3. Run `dotnet ef database update` in the terminal to create the database.
4. Press F5 or run `dotnet run` to start the app.
